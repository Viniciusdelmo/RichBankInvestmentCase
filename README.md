# 💰 RichBank Investment Case

This repository contains the **RichBank Investment API** in two implementations:

1. **[Node.js (Express) API →](NodeVersion/README.md)** — First implementation: Express 5, TypeScript, Prisma, PostgreSQL, Zod.
2. **[NestJS API →](NestVersion/README.md)** — Evolution: same domain and API contract, built with NestJS 10, class-validator, and the same Prisma/PostgreSQL stack.

Both versions offer the same features: portfolios, assets, BUY/SELL transactions, and portfolio position (holdings). Use the links above for setup, routes, and tech stack of each backend.
